# VR Tower Defense

A Virtual Reality tower defense prototype developed in Unity, focused on immersive and intuitive interaction using motion controllers.

## Overview
This project explores how VR can enhance strategy gameplay by allowing players to physically build and manage defenses in real time. Players must place structures, manage resources, and survive increasingly difficult enemy waves.

## Features
- VR-based building system using ray interaction  
- Grid-based placement with validation  
- Wave-based enemy spawning with scaling difficulty  
- Multiple enemy types (basic, fast, tank, flying)  
- Upgradeable defenses  
- Economy system (build, earn, upgrade)  
- Physical button interaction to start waves  
- Haptic and visual feedback  

## Controls (Meta Quest)
**Right Controller**
- Trigger: Place structure  
- Grip: Remove structure  
- A Button: Cycle build mode  
- B Button: Upgrade structure  
- Thumbstick: Move  

## Build Types
- Wall – Blocks and redirects enemies  
- Turret – Attacks enemies (placed on walls)  
- Trap – Damages or slows enemies  

## Objective
Defend the base from enemy waves.  
If enemies reach the goal, base health is reduced.  
Game ends when base health reaches zero.

## Running the Project
If using a Meta Quest headset with a Link Cable:
- Connect headset and enable Quest Link  
- Launch the application  
- The game will automatically display in the headset  

## Tech Stack
- Unity (URP)  
- OpenXR  
- C#  

## Notes
This is a proof-of-concept project focused on gameplay systems and VR interaction rather than visual polish.
